# TEMPERATURE-CONTROL-LED
Arduino project: Temperature-Controlled LED using DHT11 sensoR
This is a beginner Arduino project that uses a DHT11 temperature sensor to control an LED.  
The LED turns on/off based on the temperature threshold set in the code.

- Arduino Uno
- DHT11 Sensor

- Breadboard & Jumper wires
